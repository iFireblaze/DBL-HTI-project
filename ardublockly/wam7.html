    Introduction lesson 7
    In the previous lesson, you learned to turn on the LED for an amount of time and check multiple times if the button has been pressed. Then you got out of the loop and the led turned off.

    We now want the program to do this for all led and buttons.
    In this lesson, we are setting this up for all LED button combos and we learn how to create a function.


    Proceed to step 2.

    Random range
    The first important thing is that the game needs to pick a random light button combo so that the light will turn on.
    We already learned about picking a random integer for our range of loops. We can also pick a random integer for our LEDs.

    We want to let the program perform this task for the LED BUTTON 2 combo if 2 is chosen etc.
    Proceed to step 5 to learn more about the if/else funtion.


    Go to the next step.
    (4/10)
    Introducing If/else
    When we use the "if/else" block we actually let the program check the first 'if' condition then he checks the 'else if' condition and then the next 'else if' condition and so on.
    If a condition is met the code in that 'else if' is executed, and it stops checking the other conditions if no loop is implemented.
    If no condition is met he just stops the loops and does nothing.



    Go to the next step.
    (5/10)
    If/Else conditions
    For the game Wack a mole we need the if/else function to check for five conditions.

    These are:
    1. if ‘buttoncombo’ equals 1, turn on LED 1. (Checks if button 1 is pressed)

    2. else if ‘buttoncombo’ equals 2, turn on LED 2. (Checks if button 2 is pressed)

    3. else if ‘buttoncombo’ equals 3, turn on LED 3. (Checks if button 3 is pressed)

    4. else if ‘buttoncombo’ equals 4, turn on LED 4. (Checks if button 4 is pressed)

    5. else if ‘buttoncombo’ equals 5, turn on LED 5. (Checks if button 5 is pressed)

    Proceed to the next step to implement these conditions.
